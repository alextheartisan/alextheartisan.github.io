# JavaScript Types

### 6 primitives

- _Number_, which is a `double`
  - _BigInt_
- _String_ is a sequence of Unicode characters (UTF-16)
- _Symbol_
- _Boolean_
- _undefined_

### 1 non-primitive

- _Object_, has special subtype: Function, and multiple inherited subtypes: Array, , RegExp, Date, Error, etc...

### Special primitive

- _null_
