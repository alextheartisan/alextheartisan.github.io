# Wires

- Витая пара
	- CAT-5 Cable
	- RJ-45 Connector
- Оптика
- Коаксиал (legacy)

 **Пропускная способность** (bandwidth) влияет на количество одновременно посылаемых данных. Аналогия: количество полос у дороги. Аналогия для **скорости** (speed) - скоростой режим у дороги.
