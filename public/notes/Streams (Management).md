# Streams (Management)

Streams are the sequences of items / events available over time

$$
Input \xrightarrow{Process} Output
$$

- Event streams
  - Input / Output
  - Positive / Negative
- Item streams
  - Kanban
- Feedback

- Reactions to the events

  - Fragility
  - Reactivity
  - Stability

- Little's law
- Theory of constraints
  - Sankey chart

Technical analogy: RabbitMQ / CQRS

- Queueing model
