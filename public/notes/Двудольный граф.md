# Двудольный граф, биграф

aliases:
:::
Bipartite graph
:::

Все вершины двудольного графа можно разбить на две части, без граней между вершинами одной части

viz:
:::
graph {
	bgcolor = transparent
	arrowsize = .1
	node [
		shape     = circle
		width     = 0.1
		height    = 0.1
		fixedsize = true
		fontsize  = 0
		fillcolor = black
		style     = filled
	]
	edge [
		arrowsize = .6
		arrowhead = open
	]
	subgraph cluster_a {
		style = rounded
		color = lightgray
		a b c
	}
	subgraph cluster_b {
	  	style = rounded
		color = lightgray
		d e
	}
	{a b c} -- {d e}
}
:::
