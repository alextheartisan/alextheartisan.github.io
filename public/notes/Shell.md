# Shell

_Shell_ is the main way to work with the OS without GUI

## Common tasks

`stdout` redirecting:

```bash
date > file
```

`stdin` and `stdout` redirecting:

```bash
date < file1 > file2
```

Redirecting `stdout` of one program to `stdin` of another program:

```bash
date | sort
```
